## ファイルが破損している場合
本アプリは未署名のため、ダウンロード・展開したとき、macOS のセキュリティ機構（Gatekeeper）がブロックする場合があります。
その場合は以下のコマンドを実行することで、手動でアプリを認証してください。

```bash
cd path_to_ATHENAapp.app_directory
xattr -dr com.apple.quarantine ATHENAapp.app
```
